# Python Functions

## What is a Function?
A function is a block of reusable code
that performs a specific task.

Functions help in:
- Code reusability
- Better readability
- Modular programming

---

## Types of Functions
1. Built-in functions (print, len, type)
2. User-defined functions

---

## Function Syntax
def function_name(parameters):
    code block
    return value

---

## Parameters and Arguments
- Parameters: variables defined in function
- Arguments: values passed to function

---

## Return Statement
The return statement sends a value
back to the function call.

---

## Notes
Functions follow the DRY principle
(Don't Repeat Yourself).
