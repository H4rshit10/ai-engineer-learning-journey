# Python Conditional Statements

## What are Conditional Statements?
Conditional statements are used to execute different blocks
of code based on conditions.

Python uses conditions to make decisions.

---

## Types of Conditional Statements

### 1. if Statement
Executes code when the condition is true.

### 2. if-else Statement
Executes one block if condition is true,
otherwise executes else block.

### 3. if-elif-else Statement
Used to check multiple conditions.

---

## Comparison Operators Used
- ==
- !=
- >
- <
- >=
- <=

---

## Logical Operators Used
- and
- or
- not

---

## Notes
Conditional statements control the flow
of the program based on logic.
