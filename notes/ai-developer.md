# AI Developer

## Who is an AI Developer?
An AI Developer focuses on building applications
that use AI models and services to solve real-world problems.

They primarily work on integrating AI capabilities
into software products.

---

## Key Responsibilities
- Develop AI-powered applications
- Integrate AI APIs and services
- Implement chatbots and assistants
- Work with prompts and model outputs
- Collaborate with AI Engineers and product teams
- Maintain and improve AI features

---

## Skills Required
- Python or JavaScript
- Basic understanding of AI and ML concepts
- API integration
- Prompt engineering
- Familiarity with LLM platforms
- Problem-solving skills

---

## Tools Commonly Used
- Python / JavaScript
- OpenAI or other AI APIs
- Hugging Face
- LangChain
- Cloud platforms

---

## AI Developer vs AI Engineer
- AI Developer focuses more on application development
- AI Engineer focuses more on system design and optimization

Both roles often overlap in real-world projects.
